# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-07-04

### Added

- Initial release under the VibeKit VDK AI Rules branding.
- `CONTRIBUTING.md` with guidelines for new contributors.
- `RULES-SPEC.md` defining the structure for `.mdc` rule files.
- `CHANGELOG.md` to track project changes.

### Changed

- **Project Rebranding**: Migrated from DevRulesPlus/AI.rules to VibeKit VDK AI Rules.
- **Metadata Standardization**: Updated YAML frontmatter for all rule files across the `core`, `assistants`, `languages`, `stacks`, `tasks`, and `technologies` directories to conform to the new VDK specification.
- Updated `README.md` with new branding, description, and repository links.

### Removed

- Old branding and references to DevRulesPlus.
